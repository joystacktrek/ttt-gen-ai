# Specialized AI Tools Overview

## üß† Key Idea: One AI Tool Is Not Enough

Not all AI tools are designed to do the same thing.

While general chat-based AI tools can handle many tasks, **specialized AI tools are optimized for specific purposes**, such as:
- Learning from documents
- Research and fact-checking
- Writing and editing
- Creating visuals
- Organizing work and information

Knowing **which tool to use** is an important AI user skill.



## üß≠ Categories of Specialized AI Tools

Below are common categories of specialized AI tools, along with **example tools** you can explore.

You do **not** need to try them yet.



## üß† Learning & Understanding Tools

**Purpose:**  
Help you learn faster by analyzing documents such as PDFs, slides, notes, or reports.

**What these tools are good at:**
- Summarizing long documents
- Explaining complex topics in simple terms
- Answering questions based on your uploaded files

**Use these tools when:**
- You need to understand a document quickly
- You are studying or reviewing materials
- You want explanations instead of reading everything

**Example tools:**
- [NotebookLM](https://notebooklm.google.com)  
- [Humata](https://www.humata.ai)



## üîç Research & Verification Tools

**Purpose:**  
Help you find accurate information **with sources and references**.

**What these tools are good at:**
- Answering research questions
- Providing citations
- Comparing information from multiple sources

**Use these tools when:**
- Accuracy matters
- You need evidence or references
- You want to verify information

**Example tools:**
- [Perplexity](https://www.perplexity.ai)  
- [Consensus](https://consensus.app)
